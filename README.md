# 한줄 소개
2030세대를 위한 웰리빙 기록 서비스, 마음짓기 🌿

마음짓기는 성인이 되고 난 후부터 죽음에 관해 계속 기록하여 그 두려움을 줄이고, 
삶의 마무리를 준비하며 주변의 인연들에게 감사함을 전할 수 있는 경험을 제공하는 서비스입니다.
